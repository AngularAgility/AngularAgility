aa-submit-form

Description

The aa-submit-form directive is used to specify the $scope function that will be called only if the parent form is valid. If the parent form has invalid inputs, then all validation messages will appear in the form.

Overrides: 

			None.
Example: 
			<div class="form-group">
				<div class="col-sm-offset-2 col-sm-3">
					<button aa-submit-form="save()" class="btn btn-default">Save</button>
				</div>
			</div>

Dependencies

  • aaFormExtensions

Usage

as attribute
<form aa-submit-form>
   ...
</form>
