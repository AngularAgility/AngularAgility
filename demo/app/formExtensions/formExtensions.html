Must be an email address
Woah! It generated almost everything for you with only one directive!
  • aa-auto-field="person.Email" generates:
    • An automatic name="..." for the field (for use with ng-form) based on the ng-model binding
    • An overridable "aa-label" directive that follows a camel cased spacing pattern based on the ng-model binding (also 508 compliant)
    • Label generated/positioned using a configurable labelStrategy that also produced the required star *
    • A 'aa-val-msg' directive for auto generation of validation messages

This is using the "aa-submit-form" directive which is essentially a "gated" ng-click:
  • It calls the underlying scope function (just like a ng-click) ONLY if the parent form is valid
  • If clicked when the form is invalid it triggers all validation messages to appear in the form
