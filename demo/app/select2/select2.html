Select2 - Under construction!

There are a few other Select2 directives out there. This one is specifically designed to easily work with binding ids, objects, arrays, AJAX queries etc. for some pretty common use cases (see examples!)


It abstracts away the complicated Select2 API with a $settings object (lots of examples below!) and should make it pretty easy to do a lot of things.


It offers no support for the basic 'replace a <select> list and <option>' functionality. If you want that take a look at this one.


Take a look at the examples below to see what this one does readily and feel free to copy paste!

Basic Id Example


$scope.selectedStateAbbreviation = {{ selectedStateAbbreviation }}

Very basic. The bound ng-model value selectedStateAbbreviation is just the "abbreviation" for the state. Ex "MN". This is called 'id' mode by the directive since we are just storing the id.

Basic Object Example


$scope.selectedStateObject = {{ selectedStateObject | json }}

A little more complicated. ng-model is bound to "selectedStateObject" which is an object representing the currently selected state. This is called 'object' mode by the directive since we are binding full objects
